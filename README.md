greatest-common-divisor
=======================

Computes the [greatest common divisor](https://en.wikipedia.org/wiki/Greatest_common_divisor)

## Install

```sh
$ npm i -g greatest-common-divisor
```

## Usage

```js
import gcd from 'greatest-common-divisor'

gcd(12, 18) // 6

```
